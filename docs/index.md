# CogSys EEG Analysis Manual

This manual collects practical notes, tutorials, and standard operating procedures for EEG data analysis.

Start here:

- [Getting Started with EEG Analysis](eeg/getting-started.md)

## Scope

The manual is intended for onboarding students to EEG analysis workflows, including:

- EEG data loading and inspection
- basic preprocessing
- spectral analysis
- reproducible reporting
- later comparison between MNE-Python and EEGLAB workflows

## Storage note

This repository should store documentation, scripts, environment files, and small figures.

Raw EEG datasets, videos, and large binary files should be stored separately, for example on a lab server, NAS, OSF, DVC remote, or Git LFS when appropriate.
