# Start Here

???+ info "Before you begin"
    This section is for new students or collaborators who are beginning EEG analysis. Follow the pages in order: **Learning Path**, **Software Setup**, and **First EEG Practice**.

## Goal

By the end of this section, you should be able to:

- explain the basic EEG analysis workflow;
- install a working MNE-Python environment;
- load a public EEG dataset;
- inspect raw EEG signals;
- compare eyes-open and eyes-closed spectral power;
- prepare a short first report.
