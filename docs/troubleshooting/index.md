# Troubleshooting

???+ question "How to use this page"
    Start with the visible symptom, then check the most likely causes. Record the final fix so that the same issue is easier to solve next time.

## Common problems

<div class="grid cards" markdown>

-   **Noisy channels**

    ---

    Check contact quality, cap position, cable stability, and environmental noise.

-   **Missing markers**

    ---

    Check experiment software, marker cable, recording settings, and event labels.

-   **Montage errors**

    ---

    Check channel names, capitalization, standardization, and montage compatibility.

-   **PSD looks unusual**

    ---

    Check preprocessing, line noise, channel rejection, and frequency limits.

-   **MNE plotting problems**

    ---

    Check backend, notebook environment, and interactive window settings.

</div>
