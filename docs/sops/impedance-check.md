# Impedance Check

???+ warning "Draft SOP"
    Replace the example notes with the exact requirements for the recording system and study protocol.

## Checklist

- Check impedance before every recording.
- Start with reference and ground channels.
- Review channels with unstable contact.
- Adjust electrodes one by one.
- Save or record the final impedance status if the recording software supports it.

## Common notes

??? question "Why can one channel remain unstable?"
    Possible causes include dry gel, poor electrode contact, cap displacement, or a damaged electrode.
